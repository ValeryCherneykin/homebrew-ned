# homebrew-ned

Homebrew tap for [ned](https://github.com/ValeryCherneykin/ned) — open any remote file in your local editor over SSH or Docker.

## Install

```bash
brew tap ValeryCherneykin/ned
brew install ned
```

## Update

```bash
brew upgrade ned
```

## Source

Formula is automatically updated by GoReleaser on each release of [ned](https://github.com/ValeryCherneykin/ned).
